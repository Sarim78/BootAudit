# Archive

This folder contains the original prototype of the project before the v1.0.0 rewrite as BootAudit.

## Contents

- `Program.cs` — The original single-file C# console app that read the `ShutdownTime` registry value and printed the last shutdown time and a basic uptime calculation.

## Why it's kept

This is preserved for reference to show the evolution from a small registry-reading utility into a structured Windows boot forensics CLI. The original code is not maintained and should not be used. See the project root for the current version.
